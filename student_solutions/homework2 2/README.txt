How to compile?
For Problem1, open the terminal, and type in javac SymbolBalance.java. Then type in java SymbolBalance Test.java.(or Test1/2/3/4/5/6.java)
For Problem2, open the terminal, and type in javac Problem2.java. Then type in java Problem2.java.

What do they solve?
MyStack.java creates a new class using LinkedList to build a stack.

SymbolBalance.java uses stack to test whether the symbols in a file is balanced. Take into account the '"' and '/*'.

Test.java is a test file for SymbolBalance.java.

MyQueue is an interface.

TwoStackQueue implements Myqueue and completes enqueue and dequeue with only two stacks.

Problem2.java is a test file for TwoStackQueue.java.

Problem2.txt is an analysis of the big-oh running time of the enqueue and dequeue in TwoStackQueue.
