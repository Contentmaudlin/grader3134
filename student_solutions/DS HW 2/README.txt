To compile program on the command line, type:
javac SymbolBalance.java
javac MyStack.java


To run the Symbol Balance program on the command line to check a certain file named for example Test.java, type:
java SymbolBalance Test.java

Test.java can be replaced by any file to check for symbol balance.


2
To compile program on the command line, type:
javac MyQueue.java
javac Program2.java
javac TwoStackQueue.java
javac MyStack.java

To run the Two Stack Queue program, type:
java Program2


Some of the programs like MyStack.java may be automatically compiled when you compile some programs like SymbolBalance.java, but it is better to compile all of them individually.
